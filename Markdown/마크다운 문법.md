## 코드 블록

### Fenced Code block

- backtick 기호(```) 3개 활용해서 작성
- 특정 언어를 명시하면 Syntax highlighting 기능이 적용됨

```python
print('hello')

if True:
    print('t')
else:
    print('f')
```

```html
print('hello')
<h1>
    제목
</h1>
<!-- 주석 -->
```

이런 식으로 문법따라 글자 색깔이 달라짐

ctrl+enter키로 코드블록에서 바로 빠져나올 수 있음

vscode에서 바로 복사해올 수도 있음(반대도 ㅇㅋ)

### Inline Code block

`print` 함수는 파이썬에서 출력하는 함수이다.
